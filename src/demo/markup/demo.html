Options

This JumboSlider has no arrow controllers.

$("#demo-arrows").jumboslider({
    arrows: false
});

This JumboSlider has no pagination controllers.

$("#demo-pagination").jumboslider({
    pagination: false
});

This JumboSlider runs automatically. It is also able to start over from the first item after it reaches to the last one and also to jump back to the last item right from the first one. There are also custom controllers with attached JumboSlider methods.

$("#demo-loop").jumboslider({
    loop: true,
    autoplay: 3000
});
