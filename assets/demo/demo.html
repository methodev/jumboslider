Options

This JumboSlider has no arrow controllers.

$("#demo-arrows").jumboslider({
    arrows: false
});

This JumboSlider has no pagination controllers.

$("#demo-pagination").jumboslider({
    pagination: false
});

This JumboSlider has custom controller that uses the plugin's methods. It's also customized with three specific settings:

  1. 1. autoplay: It slides automatically.
  2. 2. loop: It starts over once it reaches to the last one.
  3. 3. transition: It slides faster than the default speed.
$("#demo-loop").jumboslider({
    autoplay: 3000,
    loop: true,
    transition: 200
});
